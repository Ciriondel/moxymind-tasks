Moxymind Technical Task 1 - Frontend Automation

Task Overview
Automated test suite for (https://www.saucedemo.com/) using Playwright and TypeScript.

Test Cases Included:
TC-1 Successful Login with all usernames
TC-2 Adding and Removing items from the Cart
TC-3 Complete Checkout
TC-4 Login with Performance Delay

Prerequisites
- Node.js (v16 or higher)
- npm or yarn

How to Run Locally
1. Clone the repo:
   `git clone <https://github.com/Ciriondel/moxymind-tasks.git>`
2. Install dependencies:
   `npm install`
3. Install Playwright Browsers:
   `npx playwright install`
4. Run tests in Headless mode:
   `npx playwright test`
5. Run tests in Headed mode (UI):
   `npx playwright test --headed`
